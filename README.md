# PY-Breakout:

My own version of the [Breakout](https://en.wikipedia.org/wiki/Breakout_(video_game)) game.

## Instructions:

Run `python3 main.py`

## Features:

- Multiple modes
- Multiple levels

## Controls:

- WASD or arrows: move
- Enter: Select option
- Space: Pause the game
- Mouse: Move between options and select option
